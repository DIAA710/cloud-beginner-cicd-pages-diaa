What is CI/CD?

CI/CD is a method used to automate software development and deployment processes. Continuous Integration helps developers merge and test code changes quickly and safely. Continuous Delivery and Continuous Deployment make it easier to release updates faster. This reduces manual effort, saves time, and lowers the chance of errors. In cloud computing, CI/CD improves reliability and helps keep websites and applications updated efficiently.

My Pipeline

When I push my files to the main branch on GitHub, GitHub Actions starts automatically. The workflow checks that the required project files exist, then uploads the website files and deploys them to GitHub Pages. After that, the updated website becomes available online.
