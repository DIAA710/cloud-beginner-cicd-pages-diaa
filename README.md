# Cloud Beginner CI/CD Pages

A simple static website for the Cloud Computing assignment demonstrating CI/CD deployment using GitHub Actions and GitHub Pages.

## Student Information
- **Name:** Diaa Abuhasaballah
- **Course:** Cloud Computing

## Project Description
This website was built using HTML, CSS, and JavaScript.  
It explains the concept of CI/CD and shows how GitHub Actions can automatically deploy a static website to GitHub Pages.

## GitHub Pages Link
https://diaa710.github.io/cloud-beginner-cicd-pages-diaa/

## How to Run Locally
1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.

## Files Included
- `index.html`
- `style.css`
- `script.js`
- `.github/workflows/deploy.yml`

## Deployment

This project is deployed automatically using GitHub Actions whenever changes are pushed to the `main` branch.
